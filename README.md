# ⚡ PinFlow Hub | Engineering & PCB Design

Welcome to the central repository for **PinFlow Hub**. This is where I document my journey as an **Amateur Electronics & Telecommunication Engineering Student**, moving from messy breadboards to professional-grade hardware.

> **"Turning theoretical engineering into tangible hardware, one mistake at a time."**

---

## 📟 About the Project
PinFlow Hub is built for **aspiring engineers** who want to see the real side of electronics. No "perfect" expert tutorials here—just real learning, real debugging, and real projects.

### 🛠 Current Focus
* **Embedded Systems:** Working heavily with the **ESP32-S3**.
* **PCB Design:** Transitioning from through-hole to professional **SMD layouts** using KiCad.
* **Robotics:** Developing interactive desk companions and automated systems.
