# 📟 Professional Hardware Interface Card

![Touch Card Schematic](https://raw.githubusercontent.com/urjadoshi2024etc-creator/PinFlow-Hub/567569ddcafb217990949e4e75850ec64b8a179b/TouchCard/Images/TouchCard_Sch.png)

## 🌟 Overview
This project is a sleek, professional-grade hardware interface designed as a modern alternative to bulky components. As an **E&TC student**, I designed this to master **SMD component integration** and clean PCB routing.

It features a compact, "Card-style" form factor that is perfect for integrating into custom engineering projects or as a standalone demonstration of SMD assembly.

---

## 🛠 Features
* **Driver:** BSS138 N-Channel MOSFET (High-speed switching).
* **Feedback:** 0805 SMD LED for immediate visual signal status.
* **Architecture:** Optimized for low-profile, professional applications.
* **Components:** Fully SMD (0805 footprints) for a clean, modern finish.
* **Layout:** Optimized track widths for signal integrity and heat management.

---

## 📂 Repository Structure
* **/Hardware**: Contains the original KiCad project files (`.kicad_pcb`, `.kicad_sch`).
* **/Images**: Schematic diagrams and hardware renders.
* **/Gerbers**: Production-ready ZIP files for manufacturing (JLCPCB/PCBWay).

---

## 🚀 How to Use / Order
If you want to build this yourself:
1. **Download** the `TouchCard_Gerbers.zip` from this folder.
2. **Upload** the ZIP file to any PCB manufacturer.
3. **Assemble** using the following key components:
    * BSS138 MOSFET (SOT-23)
    * 0805 Resistors & LEDs
